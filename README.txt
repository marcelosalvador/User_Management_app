Instale a versão mais atual do Node.js na sua máquina:
https://nodejs.org/en/download

Depois de tudo instalado corretamente,
em seu prompt, navegue até o diretório da aplicação Angular

Digite os seguintes comandos:
npm install
npm start

Você deverá receber uma mensagem se estiver tudo certo:
√ Compiled successfully.

No seu navegador preferido entre no link:
http://localhost:4200/
